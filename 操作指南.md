# LifeBubble 快速启动指南

## 🎯 方法一：使用单文件版本（最简单）

### 1. 在 Xcode 中打开你的项目
- 双击打开你的 `dream.xcodeproj` 项目

### 2. 删除所有旧代码文件
在 Xcode 左侧导航栏中：
- 找到并**右键点击** `dreamApp.swift`（或任何带有 @main 的文件）
- 选择 **Delete** → **Move to Trash**
- 同样删除 `ContentView.swift`
- **保留** `Assets.xcassets` 文件夹

### 3. 导入新的完整文件
- 打开 Finder，进入 `/Users/xinyao/Desktop/dream/`
- 找到 **`CompleteApp.swift`** 文件
- **拖拽**这个文件到 Xcode 左侧的项目导航栏中
- 在弹出的对话框中：
  - ✅ 勾选 **Copy items if needed**
  - ✅ 勾选 **Create groups**
  - ✅ 确保 **dream** 被勾选（在 Add to targets 下）
  - 点击 **Finish**

### 4. 清理并运行
- 按 `Shift + Cmd + K`（清理构建）
- 按 `Cmd + R`（运行）
- 等待编译完成

### 5. 查看效果
应用应该启动并显示：
- ✅ 呼吸光球（启动页）
- ✅ 长按1秒进入主页
- ✅ 看到5个漂浮的泡泡
- ✅ 所有交互都能正常工作

## 🎯 方法二：完全重新开始（如果方法一不行）

### 1. 关闭当前项目
- 在 Xcode 中：`File` → `Close Project`

### 2. 创建全新项目
- 打开 Xcode
- 点击 **Create a new Xcode project**
- 选择：**iOS** → **App**
- 点击 **Next**
- 配置：
  - Product Name: `LifeBubble`（或任意名称）
  - Team: 选择你的开发者账号
  - Organization Identifier: `com.yourname`
  - Interface: **SwiftUI**
  - Language: **Swift**
  - **取消勾选** 所有其他选项
- 点击 **Next**，保存到桌面

### 3. 删除默认文件
- 在新项目中，删除默认的：
  - `LifeBubbleApp.swift`（或项目名+App.swift）
  - `ContentView.swift`

### 4. 导入 CompleteApp.swift
- 将 `/Users/xinyao/Desktop/dream/CompleteApp.swift` 拖入项目
- 勾选 **Copy items if needed**

### 5. 运行
- 选择目标设备（你的 iPhone 或模拟器）
- 按 `Cmd + R` 运行

## 🔧 常见问题排查

### 问题1：编译错误 "Multiple @main"
**原因**：项目中有多个入口文件

**解决**：
1. 在 Xcode 搜索框（Cmd + Shift + F）搜索 `@main`
2. 确保**只有一个**文件包含 `@main`
3. 删除其他有 `@main` 的文件

### 问题2：还是显示 "Hello World"
**原因**：缓存问题

**解决**：
1. `Product` → `Clean Build Folder` (Shift + Cmd + K)
2. 关闭 Xcode
3. 在手机/模拟器上删除应用
4. 重新打开 Xcode
5. 再次运行

### 问题3：找不到 Color(hex:)
**原因**：文件不完整

**解决**：
确保使用的是 `/Users/xinyao/Desktop/dream/CompleteApp.swift`
这个文件在最底部包含了 `Color` 扩展代码

### 问题4：应用闪退
**原因**：代码有错误

**解决**：
1. 查看 Xcode 底部的错误日志
2. 确保文件完整导入
3. 尝试在模拟器上运行（而不是真机）
